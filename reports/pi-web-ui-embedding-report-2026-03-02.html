Risk / Environment Notes

Unusual Whales access remains required for flow-driven commands through `UW_TOKEN`. Commands are strict-whitelisted and reject unknown or positional injection patterns for production safety. Command execution is bounded by a timeout and outputs are normalized to avoid unbounded payloads.
