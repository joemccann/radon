Step-by-step Instructions
  1. Open terminal and change to the project root:

    cd /Users/joemccann/dev/apps/finance/convex-scavenger

  2. Go into the web app directory:

    cd web

  3. Install dependencies:

    npm install

  4. Create environment file:

    cp .env.example .env

  5. Set required keys in web/.env:
    • Set one Anthropic key: ANTHROPIC_API_KEY or CLAUDE_CODE_API_KEY or CLAUDE_API_KEY
    • Set UW_TOKEN for UW-backed PI commands
    • Optional: ANTHROPIC_MODEL, ANTHROPIC_API_URL
  6. Start development server:

    npm run dev

  7. Open UI in browser:

    http://localhost:3000

  8. (Optional) Validate with tests:

    ASSISTANT_MOCK=1 npm test

  9. (Optional) Production start:

    npm run build

    npm run start

Operational Checklist

Before first real run: confirm UW_TOKEN is valid and accessible in the same shell context as Next.js.

If commands fail immediately in assistant mode, confirm AI keys and API route connectivity in web/.env.

If Python command route is expected to work, confirm scripts are present and executable from repo root via the server process working directory.
